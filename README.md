  Projeto Fila de Atendimento

Sistema simples de gerenciamento de fila de atendimento desenvolvido em Java, com o objetivo de simular o funcionamento de filas em ambientes como bancos, clínicas e atendimentos em geral.

  Sobre o projeto

Este projeto tem como finalidade aplicar conceitos fundamentais de programação, como:

- Estruturas de dados (Fila / Queue)
- Orientação a Objetos (POO)
- Organização lógica de fluxo de atendimento

Sistemas de fila são amplamente utilizados para organizar o atendimento e garantir ordem de chegada ou prioridade, tornando o processo mais eficiente .

  Funcionalidades
  
- Adicionar pessoas à fila
- Remover (atender) o próximo da fila
- Visualizar a fila atual
- Simulação de atendimento sequencial
- 
  Tecnologias utilizadas
- Java
- Paradigma Orientado a Objetos

  Como executar o projeto
  
- Clone o repositório:
- git clone https://github.com/PedroG1543/Projeto-fila-de-atendimento
- Acesse a pasta do projeto:
- cd Projeto-fila-de-atendimento
- Compile os arquivos:
- javac Main.java
- Execute o programa:
- java Main
  
  Conceitos aplicados

Este projeto utiliza o conceito de fila (FIFO - First In, First Out), onde o primeiro elemento a entrar é o primeiro a sair.

  Objetivo

O principal objetivo deste projeto é:

- Praticar lógica de programação
- Consolidar conceitos de estruturas de dados
- Servir como projeto de portfólio para desenvolvimento Java
